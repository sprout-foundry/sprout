- [ ] SP-046-1: **Sync protocol + per-file metadata.** Add `pkg/agent_tools/workspace_sync.go` defining `FileMetadata { browser_seq, container_seq, last_synced_browser, last_synced_container, modified_at }` and the patch event payload `{ path, container_seq, content, base_browser_seq }`. Define the WS envelope (`workspace.patch_in`, `workspace.patch_out`, `workspace.heartbeat`) and the HTTP `POST /api/workspace/op` outbound endpoint in `pkg/webui/workspace_sync_handlers.go`. Acceptance: a Go test drives the protocol end-to-end (container emits patch → browser acks with `last_synced_container`; browser emits op → container applies + bumps `container_seq`); the per-file sequence rules from spec §3 are honored.
