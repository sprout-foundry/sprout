## SP-050: Orchestrator Persona Collapse — One Persona, Configurable Git-Write
_Spec: `roadmap/SP-050-orchestrator-persona-collapse.md`_

Collapse the near-identical `orchestrator` and `repo_orchestrator` personas into a single `orchestrator` persona. Git-write capability follows from the existing `AllowOrchestratorGitWrite` config flag instead of a separate persona ID. Single phase, single PR.

Embed it into the binary via a new `go:embed` directive co-located with the existing prompt embeds.
