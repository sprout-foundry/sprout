# @sprout/ui

## 0.1.0

Initial release with extracted components:
- Editor (CodeMirror 6-based)
- Terminal (xterm.js-based)
- FileTree (with multi-select, drag-and-drop, fuzzy filter)
- GitPanel / GitSidebarPanel
- ChatPanel / MessageBubble / MessageContent / LiveLog
- Sidebar, CommandPalette, StatusBar
- NotificationStack, ContextMenu
- SproutProvider / SproutAdapterContext (adapter pattern)
- EventsContextProvider (events abstraction)
- Utility hooks: useMultiSelect
- Utility functions: fuzzyMatch, clipboard, uuid, ansi, etc.
