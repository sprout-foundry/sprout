# Docs Sync Workflow

Use this skill when behavior, commands, or APIs change.

## Steps

1. Identify behavior changes that affect users/developers.
2. Update relevant docs and examples.
3. Remove stale instructions.
4. Verify command snippets and paths are accurate.

## Required Output

- Docs updated with changed behavior
- Removed stale/outdated sections
- Validation of key examples/commands
