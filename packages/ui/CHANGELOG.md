# Changelog

All notable changes to `@sprout/ui` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-04-29

### Added

- Initial release of `@sprout/ui` as a publishable package.
- React components: Editor, Terminal, FileTree, Sidebar, CommandPalette, StatusBar, ContextMenu, GitPanel, ChatPanel, and more.
- Contexts: NotificationProvider, SproutProvider, EventsContextProvider.
- Hooks: useMultiSelect, useNotifications, useSproutAdapter, useSproutFetch, useEvents.
- Utilities: fuzzy matching, clipboard, UUID generation, line ending detection.
- Dual ESM/CJS output with TypeScript declarations.
